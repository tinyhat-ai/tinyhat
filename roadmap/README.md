# Tinyhat Plugin Roadmap

This folder records public roadmap ordering for the
`tinyhat-ai/tinyhat` OpenClaw plugin package.

The roadmap is not a deadline list. It is a priority view of issues
that shape the public plugin package: manifest, tools, default skills,
validation, docs, and release metadata.

| File | Meaning |
| --- | --- |
| [`now.md`](now.md) | 1-3 items actively being worked on |
| [`next.md`](next.md) | queued after `now` clears |
| [`later.md`](later.md) | larger bets not yet scheduled |
| [`considering.md`](considering.md) | ideas being evaluated |
| [`rejected.md`](rejected.md) | proposals explicitly declined |

Every committed roadmap item should link to a GitHub issue. The issue
owns the detailed spec; this folder owns sequence and priority.
