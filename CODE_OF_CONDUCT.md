# Code of Conduct

This project adopts the **Contributor Covenant, version 2.1** as its
code of conduct.

The full text is published at:

- <https://www.contributor-covenant.org/version/2/1/code_of_conduct/>
- <https://www.contributor-covenant.org/version/2/1/code_of_conduct.md> (markdown)

By participating in this project (contributing code, opening issues,
reviewing PRs, commenting in discussions) you agree to abide by it.

## Reporting

If you believe someone is violating the Code of Conduct, contact the
project maintainer privately:

- Email: **conduct@tinyloop.co**

All reports will be reviewed and investigated promptly and fairly. The
maintainer will respect the reporter's privacy and will not disclose
identifying details without consent, beyond what's necessary to act on
the report.

## Scope

This Code of Conduct applies within all project spaces — the GitHub
repository, issue tracker, pull requests, discussions — and in public
spaces when an individual is representing the project.

## Enforcement

Instances of abusive, harassing, or otherwise unacceptable behavior may
result in temporary or permanent removal from the project's community
spaces, at the maintainer's discretion. The enforcement ladder and
community-impact guidelines are those described in the Contributor
Covenant v2.1 linked above.
